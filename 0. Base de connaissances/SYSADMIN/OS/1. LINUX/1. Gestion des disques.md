# 1. Créer partition

1. Ajouter le disque manuellement sur la vm
2.  Reperer disque avec lsblk

```
lsblk
```

3. Partitionner le disque

```
fdisk /dev/<sdx>
```

4. n -> entré -> entré -> entré -> w

# 2. Formatter le disque

```
mkfs.ext3 /dev/<sdx>
```
