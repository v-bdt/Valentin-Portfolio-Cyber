1.  Nettoyer fichiers de packages en cache

```
sudo apt-get update && sudo apt-get autoclean && sudo apt-get clean && sudo apt-get autoremove
```

2. Afficher espace disque

```
df -h
```

3. Installer ncdu

```
apt install ncdu
```

3. Afficher la tailles des dossiers en detail

```
ncdu /
```
